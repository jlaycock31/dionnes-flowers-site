GitHub Connection

This connects the admin to your website's files on GitHub. You only need to do this once.

  1. Go to github.com → click your profile photo → Settings
  2. Scroll to Developer Settings → Personal Access Tokens → Tokens (classic)
  3. Click Generate new token (classic)
  4. Give it any name, check the repo box, click Generate
  5. Copy the token below — save it somewhere safe, GitHub only shows it once!
